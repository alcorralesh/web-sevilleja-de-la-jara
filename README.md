# Portal vecinal · Sevilleja de la Jara

Prototipo funcional y responsive del portal del pueblo.

## Abrir la web

Abre `index.html` en cualquier navegador moderno. No necesita instalación.

## Acceso de demostración

- Usuario: `admin`
- Contraseña: `Sevilleja2026`

Desde el área privada se pueden publicar artículos (incluyendo imágenes), crear y editar eventos, consultar o exportar las inscripciones y llevar los ingresos y gastos.

## Persistencia

Esta versión guarda los datos en el navegador (`localStorage`), por lo que funciona como prototipo completo en un solo dispositivo. Para publicarla con varios usuarios reales hace falta conectar el frontal a un servidor y una base de datos, además de sustituir el acceso de demostración por autenticación segura.
